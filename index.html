關於我

哈囉！我是陳兆臨，目前就讀於國立臺北科技大學智慧自動化工程系五年一貫學程二年級。 我喜歡在不同領域中挑戰自己學習，從前端開發到資訊安全，甚至是3D建模與語言學習都是我探索的範圍。 透過不斷的練習與嘗試，我希望能將程式設計、創意與工程思維結合，創造出兼具功能性與美學的作品，並持續拓展自己的可能性。

專案

個人網站建置

這個網站就是我的第一個主要專案，透過 HTML, CSS 和 JavaScript 從零開始建置， 學習了網頁基本結構、樣式設計與互動功能。

利用 Fusion 360 進行多項零件設計與組裝，並結合 3D 列印技術將設計實體化， 從中學習了產品設計流程與機械結構。

基礎資訊安全實踐

參與資訊安全社團並學習基本資安概念，包含弱點掃描、滲透測試基礎等， 提升了對網路安全的認識與實作能力。
