關於我

哈囉！我是陳兆臨，目前就讀於國立臺北科技大學五專智慧自動化工程科，現在是專二學生。我喜歡挑戰不同領域的學習，從前端開發到資訊安全，甚至 3D 建模與語言學習，都是我探索的方向。透過不斷實作與嘗試，我希望能結合程式設計、創意與工程思維，打造兼具功能與美感的作品，並持續拓展自己的可能性。

專案

這個網站就是我的第一個主要專案，透過 HTML, CSS 和 JavaScript 從零開始建置，學習了網頁基本結構、樣式設計與互動功能。

目前正持續學習 資安相關知識，包含基礎網路安全概念、弱點掃描與滲透測試原理等。雖然仍在進修階段，但透過課程與自學逐步累積對資訊安全的理解，未來希望能在實務專案中應用所學。
